HDA Plastic Surgery Face Database
=================================

Facial images taken before face surgeries are marked with '_b' and images taken after face surgeries are marked with '_a'

Download and Copyright:
The database is being made available for the researchers from 2020 onwards.  Any commercial use/distribution of this database is prohibited. All the technical reports and papers that report experimental results from this database should provide due acknowledgment and reference to [1]. 

References
[1] Christian Rathgeb, Didem Dogan, Fabian Stockhardt, Maria De Marsico, Christoph Busch, „Plastic Surgery: An Obstacle for Deep Face Recognition?“, in 15th IEEE Computer Society Workshop on Biometrics (CVPRW), pp. 3510-3517, 2020.
